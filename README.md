# Hive Bingo
A Bingo Board made using json, compatible with the website / browser source for [lockout.live](https://beta.lockout.live/editor)

## To Use:
1. Import the .json file in this repo to this website: [lockout.live](https://beta.lockout.live/editor)
2. In settings, copy the browser source link to your OBS in a "browser source"
3. On the newly created browser source, click "interact"
4. Customize the settings to your liking, then send your friend the browser source link (they can use a browser or the interactable browser source method as well)
5. Have fun!
